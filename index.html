👨‍💻 ประสบการณ์และความเชี่ยวชาญ

ผมเป็น UX/UI Designer และ Frontend Developer ที่หลงใหลในการออกแบบและพัฒนาเว็บแอปพลิเคชันที่มอบประสบการณ์ผู้ใช้ที่ราบรื่นและมีคุณค่า ด้วยความสามารถในการเชื่อมโยงระหว่าง ดีไซน์ที่สื่อสารได้อย่างมีประสิทธิภาพ และ เทคโนโลยีที่ทันสมัย ผมสามารถแปลงความต้องการของผู้ใช้งานให้กลายเป็นผลิตภัณฑ์ที่ใช้งานได้จริงและตอบโจทย์

ผมมีความถนัดเป็นพิเศษในการออกแบบและพัฒนา ระบบ Dashboard และ Web Application เพื่อการจัดการ (Management Systems) ที่ตอบสนองการใช้งานของผู้ดูแลระบบและผู้ใช้งานในระดับองค์กร เช่น ระบบบุคลากร ระบบจัดเวร ระบบคลังข้อมูล ระบบคลังสินค้า ระบบงานพิธี ระบบลงทะเบียนผู้ป่วยและติดตามอาการ ระบบบริหารโครงการราชการ ไปจนถึงระบบบริหารภายในหน่วยงานราชการ โดยให้ความสำคัญกับการจัดโครงสร้างข้อมูล การนำเสนอภาพรวม และการใช้งานที่ง่ายต่อผู้ใช้จริง

มีประสบการณ์เชิงลึกในการพัฒนาเว็บแอปด้วย Nuxt 2, Vuetify 2, และ VueBootstrap รวมถึงการออกแบบ UX/UI ด้วย Figma ตั้งแต่กระบวนการ UX Research, Wireframing, Prototyping จนถึงการส่งต่อดีไซน์ให้ทีมพัฒนา

ผมเชื่อในแนวคิด "Design with Purpose" — ทุกองค์ประกอบในงานออกแบบต้องมีเหตุผล และทุกโค้ดที่เขียนต้องสร้างประโยชน์ให้กับผู้ใช้

ระบบรับสมัคร

แพลตฟอร์มจัดการการรับสมัครออนไลน์แบบครบวงจร ออกแบบมาให้ใช้งานง่าย รองรับทั้งเจ้าหน้าที่และผู้สมัคร พร้อมระบบแจ้งเตือนและติดตามสถานะการสมัครแบบเรียลไทม์

คุณสมบัติเด่น:

  • ลงทะเบียนออนไลน์ได้สะดวกผ่านทุกอุปกรณ์
  • ระบบอนุมัติหลายขั้นตอน
  • รองรับบทบาทผู้ใช้งานที่หลากหลาย
  • ส่งอีเมลแจ้งเตือนอัตโนมัติเมื่อสถานะเปลี่ยน
  • ระบบ Dashboard สำหรับผู้ดูแลเพื่อติดตามและวิเคราะห์ข้อมูลการสมัคร

ระบบคลังพัสดุ

ระบบบริหารจัดการพัสดุและครุภัณฑ์ภายในหน่วยงานแบบดิจิทัล ช่วยลดงานเอกสารและเพิ่มความแม่นยำในการติดตามพัสดุ ตั้งแต่การจัดซื้อ การเบิกจ่าย การยืมคืน ไปจนถึงการตรวจสอบคงคลังประจำปี

คุณสมบัติเด่น:

  • จัดเก็บข้อมูลพัสดุแบบแยกหมวดหมู่ ค้นหาได้ง่าย
  • ระบบเบิก-จ่าย และอนุมัติออนไลน์ ลดภาระงานเจ้าหน้าที่
  • รองรับการแจ้งเตือนเมื่อพัสดุใกล้หมดหรือเกินรอบตรวจสอบ
  • Dashboard สรุปสถานะพัสดุ คงเหลือ และประวัติการเคลื่อนไหวแบบเรียลไทม์
  • Export รายงานเป็น Excel หรือ PDF ได้ตามแบบฟอร์มราชการ

Tech Stack:

Nuxt 2 Vuetify 2 Figma

ระบบปลูกต้นไม้ออนไลน์

แพลตฟอร์มเชิงอนุรักษ์ที่เปิดโอกาสให้บุคลากรภายในหน่วยงาน ครอบครัว หรือประชาชนทั่วไป สามารถเข้ามาลงทะเบียนเพื่อร่วมเป็นส่วนหนึ่งของกิจกรรมปลูกต้นไม้ และติดตามผลการเติบโตได้อย่างมีส่วนร่วม

คุณสมบัติเด่น:

  • ลงทะเบียนออนไลน์เพื่อเข้าร่วมกิจกรรมปลูกต้นไม้ในพื้นที่ที่กำหนด
  • เลือกชนิดต้นไม้ที่ต้องการปลูก และระบุชื่อผู้ร่วมกิจกรรม
  • ติดตามสถานะต้นไม้ที่ปลูก พร้อมแนบภาพถ่ายการเจริญเติบโต
  • ระบบออกใบประกาศนียบัตร/เกียรติบัตรอัตโนมัติ สำหรับผู้เข้าร่วมกิจกรรม
  • แผนที่แสดงจุดปลูก และสถิติการมีส่วนร่วมในแต่ละพื้นที่

ระบบจัดการข้อมูลผู้ป่วย

ระบบบริหารจัดการข้อมูลผู้ป่วยแบบครบวงจร ออกแบบมาเพื่อรองรับการติดตามการรักษา การจ่ายยา และการประเมินอาการอย่างต่อเนื่อง เหมาะสำหรับหน่วยงานด้านสาธารณสุขหรือโครงการพิเศษที่ต้องดูแลผู้ป่วยเฉพาะกลุ่ม

คุณสมบัติเด่น:

  • บันทึกข้อมูลผู้ป่วยรายบุคคล พร้อมประวัติการรักษาและอาการ
  • ระบบบันทึกอาการแรกรับ และติดตามอาการรายวัน
  • แยกสิทธิ์ผู้ใช้งานตามบทบาท เช่น แพทย์ พยาบาล เจ้าหน้าที่บันทึกข้อมูล
  • ระบบ RBAC (Role-Based Access Control)
  • รองรับการแนบภาพถ่ายเอกสาร/ผลตรวจ และการอัปเดตเรียลไทม์
  • ดูสถิติการรักษา และข้อมูลผู้ป่วยรายวันได้ผ่าน Dashboard

พระปรมาภิไธย

ระบบ Mobile Webview สำหรับแสดงพระปรมาภิไธยของพระมหากษัตริย์และพระบรมวงศานุวงศ์ทุกพระองค์ ครอบคลุมทั้งรูปแบบเต็มและรูปแบบย่อ พร้อมคำอ่านและระบบอ่านออกเสียง รองรับการแสดงผลทั้งภาษาไทยและภาษาอังกฤษ

คุณสมบัติเด่น:

  • รองรับการใช้งานผ่านอุปกรณ์พกพา (Mobile Webview)
  • แสดงพระปรมาภิไธยครบถ้วน ทั้งรูปแบบย่อและแบบเต็ม
  • มีระบบแสดงคำอ่าน (ถอดเสียง) พร้อมระบบ Text-to-Speech สำหรับอ่านออกเสียงพระปรมาภิไธย
  • แสดงพระปรมาภิไธยในภาษาอังกฤษตามรูปแบบทางราชการ
  • ระบบค้นหาชื่อพระองค์ พร้อมการจัดหมวดหมู่ตามราชวงศ์/พระอิสริยยศ
  • อินเทอร์เฟซเรียบง่าย อ่านง่าย เหมาะสำหรับงานศึกษา งานพิธี หรือการใช้งานในหน่วยงานราชการ

คำราชาศัพท์

ระบบ Mobile Webview สำหรับรวบรวมและให้ความรู้เกี่ยวกับคำราชาศัพท์ ออกแบบให้เข้าถึงง่ายทั้งบนมือถือและแท็บเล็ต พร้อมฟีเจอร์การค้นหาและการจัดหมวดหมู่เพื่อการเรียนรู้ที่สะดวกและเป็นระบบ

คุณสมบัติเด่น:

  • รองรับการใช้งานผ่านอุปกรณ์พกพา (Mobile Webview)
  • แสดงคำราชาศัพท์ใน 2 รูปแบบ:หน้ารวม / แบบจัดหมวดหมู่
  • ระบบค้นหาคำแบบเรียลไทม์ ค้นจากคำทั่วไปหรือคำราชาศัพท์ก็ได้
  • แสดงคำแปล ความหมาย และบริบทในการใช้งานอย่างชัดเจน
  • อินเทอร์เฟซเรียบง่าย อ่านง่าย เหมาะสำหรับงานศึกษา งานพิธี หรือการใช้งานในหน่วยงานราชการ
